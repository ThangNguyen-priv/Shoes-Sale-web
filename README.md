# 🛒 Shoe Store Web App

Một ứng dụng web bán giày đơn giản với **ASP.NET Core (Backend)** và **HTML/CSS/Bootstrap 5 (Frontend)**.

---

## 📦 Tech Stack

| Layer      | Công nghệ sử dụng        |
|------------|--------------------------|
| Backend    | ASP.NET Core (.NET 6+)   |
| Frontend   | HTML, CSS, Bootstrap 5   |
| Database   | SQL Server |
| ORM        | Entity Framework Core    |
